# Irrigation-Predictor-App
In this app, you can use sliders to set the values for parmeters like wind speed, temparature and drop down box for categorical features. After setting parameter click on predict to 
get the irrigation need based on the parameters you set
## Note
Model files (pkl) not included due to size. 
To run locally, train the model using the 
notebook and save pickle files using the 
instructions in the notebook.
